# Solution Architecture

The architecture of the application is built on a modular design which helps in scaling and maintenance. Each module is designed to handle specific features, which facilitates a clear separation of concerns.

## Features
- User Authentication
- Data Visualization
- Real-time Updates
- Multi-language Support

## Development Server Setup

To set up the development server, follow these instructions:
1. Clone the repository: `git clone https://github.com/vaibhavkadam2006/Dermisol.git`
2. Navigate into the project directory: `cd Dermisol`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`

Make sure to have Node.js installed on your machine before proceeding with the installation of dependencies.
